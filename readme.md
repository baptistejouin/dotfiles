My current config files, basically the essential tools I use daily. After years of refining, I’ve settled on a fine-grained workflow that fits my life as a software engineer and coding/art guy.

Key tools:

- [Aerospace](https://github.com/nikitabobko/AeroSpace): Window and tile manager (without navigating between more than 3 windows is absolute chaos).
- [Helix](https://github.com/helix-editor/helix): My main code and text editor (no mouse needed).
- [Ghostty](https://github.com/ghostty-org/ghostty): Terminal emulator, split between Helix and running [cowsay](https://en.wikipedia.org/wiki/Cowsay).
- [Karabiner](https://github.com/pqrs-org/Karabiner-Elements): Makes Escape easier to hit (as accidental Caps Lock are more frequent than actual Caps Lock).
- [Git](https://github.com/git/git): Because destructive work is as bad as destructive edits in Photoshop.
- [Helium](https://github.com/imputnet/helium): No config, but mandatory—no noise, maximum privacy, the browser as it should be.
- [Raycast](https://www.raycast.com/): With 2-3 extensions, basically the app that launches all other apps.
